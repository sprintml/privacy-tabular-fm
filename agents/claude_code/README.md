# Claude Code (Opus 4.7) agentic baseline

Harness for running Claude Code with Claude Opus 4.7 as an autonomous
privacy-attack agent against the competition's TFM targets. The agent
operates in a sandboxed environment with GPU access and a fixed time
budget, writes attack code from scratch given a natural-language task
specification, and produces a predictions file scored by the same
evaluation harness used for participant submissions.

Released as a frozen, reproducible artifact per the competition's
"Agentic baselines" paragraph (proposal §1.6).
