# Data preparation

Scripts that download and preprocess the three reference datasets
used as the distribution reference for TabSyn synthetic-data
generation.

- `prepare_folktables.py` — Folktables ACSIncome (California, 2018).
- `prepare_adult.py` — UCI Adult Income.
- `prepare_texas.py` — Texas Hospital Inpatient Discharge PUDF.
  Pending data-use approval from the Texas Department of State
  Health Services.

TabSyn training scripts and the synthetic-record sampling pipeline
will be added here ahead of the competition launch.
