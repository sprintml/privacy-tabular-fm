# TASK 3: Dataset Inference

Dataset Inference (DI) determines whether a corpus was used during the training of a target model. Unlike membership inference, which evaluates individual records, DI aggregates membership signals across many records into a dataset-level statistic, amplifying weak per-record evidence into a statistically significant
claim. The track targets a TFM conditioned on a private in-context training set. Participants are provided with the target model and a candidate suspect dataset and asked to determine whether the suspect dataset contributed to the model's in-context training set. We provide [N] target models, each conditioned on one of [K] candidate corpora drawn from the underlying data distribution. For each (target model, candidate corpus) pair, participants submit a probabilistic decision of training inclusion. Real-world analog: a data owner suspects that their tabular corpus has been used as the in-context training set of a publicly deployed model without consent; DI provides a statistical procedure to support such a claim.
